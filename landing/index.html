The Stack

Four services. One command.

Every component purpose-built, every integration pre-wired. Deploy the full team or pick what you need.

⚡
OpenClaw

Agent gateway that bridges your AI team to the channels your users already live in. Route messages, manage sessions, and fan out across platforms from a single config.

Discord · Telegram · Slack
🦀
OpenFang

Autonomous agent operating system written in Rust for raw speed and safety. Manages agent lifecycles, tool dispatch, sandboxed execution, and concurrency at the OS level.

Rust-powered · Zero overhead
📎
Paperclip

Workflow orchestration layer with visual org charts, cron scheduling, and multi-agent pipelines. Define how agents collaborate, delegate, and report — no code required.

40+ built-in tools · Cron + triggers
🧠
Hermes

Cognitive agent by Nous Research. Persistent memory, skill acquisition, and continuous learning mean your agents get smarter with every interaction — entirely on your hardware.

Nous Research · Persistent memory
Capabilities

Everything your agents need.

From model routing to sandboxed execution — all included, zero configuration debt.

🔌
Multi-provider LLMs

Route to OpenAI, Anthropic, Mistral, Groq, Ollama, or any OpenAI-compatible endpoint. Swap models per agent, per task, or per cost threshold — without touching agent code.

💾
Persistent memory

Hermes gives agents long-term memory that survives restarts. Episodic, semantic, and working memory — all indexed and retrievable.

⚙️
Workflow automation

Paperclip cron jobs and event triggers let agents act on schedules, webhooks, or each other's output with full audit trails.

📡
Cross-platform channels

OpenClaw connects your agents to Discord, Telegram, and Slack out of the box. Add custom HTTP adapters to reach any surface — your app, internal tools, or mobile clients.

🔒
Sandboxed execution

OpenFang isolates every tool call. Runaway processes, file system access, and network calls are scoped and audited by default.

🛠️
Skills marketplace

40+ built-in tools ship with the stack. Web search, code execution, file I/O, API calls — installed and wired to every agent on first boot.

🖥️
Unified dashboard

ClawHQ Dashboard gives you a single pane of glass: agents, workflows, channels, memory, and logs — all live, all interactive.

Quickstart

From zero to running
in 2 minutes.

Three commands. One Docker stack. Your entire AI team, online.

01
Clone the repo

Grab the full platform from GitHub. All services, configs, and defaults included.

git clone https://github.com/MJFlanigan5/clawhq-platform
02
Run the installer

The install script sets up your env files, generates secrets, and validates dependencies.

./install.sh
03
Launch the stack

All four services start together. Dashboard available at localhost:3000.

docker compose up -d
