Security

Built-in AI agent defense.
Six layers deep.

AI agents face threats that traditional security tools weren't built for. ClawHQ ships with Sentinel — a purpose-built, always-on security system that runs automatically inside every deployment.

01
Gate
Rate limiting, IP reputation, and auth — the perimeter.
02
Prompt Guard
Detects and blocks prompt injection, jailbreaks, system prompt extraction, and indirect injection via tool outputs.
03
Tool Guard
Sandboxes file access, blocks dangerous terminal commands, and catches data exfiltration via tool chains before they execute.
04
Canary Engine
Fake API keys, URLs, and credentials planted in the agent environment. If any are used externally, you have confirmed compromise — not a guess.
05
Decoy Swarm
Honeypot agents in the registry — admin-agent, db-agent, debug-agent. Attackers find them and probe them. You get alerts. They get synthetic noise.
06
Shield
Automated response: block IPs, terminate sessions, enter lockdown, capture forensic snapshots. Alerts to Discord, Slack, email, or PagerDuty.
